#### 题目描述
求1+2+3+...+n，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C）。
```c++
//利用构造函数求解
class A {
private:
    static int n;
    static int sum;
public:
    A() {++n; sum += n;}
    static void reset() {n = 0; sum = 0;}
    static int getSum() {return sum;}
};
int A::n = 0;
int A::sum = 0;
class Solution {
public:
    int Sum_Solution(int n) {
        A::reset();
        A *a = new A[n];
        delete []a;
        a = NULL;
        return A::getSum();
    }
};
```
